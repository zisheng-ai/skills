# Distribution Launcher

Use this reference during phase 7 to create a safe initial traffic plan.

## Safe Channels

Choose channels where users have real interest:

- Google organic search
- Reddit or niche forums
- Quora or Stack Exchange style Q&A where allowed
- Pinterest for visual/template topics
- YouTube Shorts or TikTok demos
- Product Hunt, Indie Hackers, Hacker News, or dev communities for developer tools
- Directory submissions
- Newsletter or creator outreach
- Small paid search or social tests for legitimate users

## Unsafe Channels

Reject:

- Traffic packages promising guaranteed visits
- Bot visits
- Autosurf and traffic exchange networks
- Incentivized ad clicks
- Click rings
- Popunder traffic for AdSense approval
- Any plan whose main purpose is inflating impressions instead of reaching users

Prioritize signals that indicate real utility: returning users, tool completions, organic queries, saves, comments, backlinks, and natural shares.
